Section 0

Introduction to HTML

HTML - HyperText Markup Language

  • gives structure to the page
  • “A markup language is a computer language that defines the structure and presentation of raw text. Markup languages work by surrounding raw text with information the computer can interpret, "marking it up" to be processed.”
  • <!DOCTYPE html> lets the computer know that we are going to be giving it HTML in the document, and declares the specific version of html that we will be using
  • <html>. </html> lets the browser know that the enclosed code is HTML
  • an element is a whole block enclose by tags
  • the tag is the element name surrounded by angle brackets

Important Tags


  • <html>
  • <head>
  • <title>
  • <p>
  • <h1> <h2> <h3> <h4>
  • <ul> and <ol> - unordered lists and ordered lists
  • <li>
  • <br> (Self closing)
  • <a href="url">link text</a>
  • <div> - defines an arbitrary "division" (i.e. section) of the html file which can then have CSS or JavaScript applied to it
  • <img src="swimmingCat.jpg" alt="Swimming Cat">

Introduction to CSS

CSS - Cascading Style Sheets

  • gives style to the page
  • Called "cascading" style sheets because of priority scheme that determines which rule applies to a given element if multiple apply (sort of like inheritance)
  • CSS can be external, internal, or in-line
  • CSS Box Model
