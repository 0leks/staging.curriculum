Typing Tutor


We are going to make a Typing Tutor.




1. Make a typingTutor class with a main method and a setup() method. Create a JFrame and a JPanel in setup method. 

 void setup(){
         JFrame frame = new JFrame();
         JPanel panel = new JPanel();
         }

2. Create a member variable in class. 

  char currentLetter;


3. Initialize currentLetter in the setup method as shown below: 

 currentLetter = generateRandomLetter();

4. Paste the following method after the main method. 

char generateRandomLetter() {
    Random r = new Random();
    return (char) (r.nextInt(26) + 'a');
}

5.Create a JLabel and set the text of the label to the currentLetter in the setup method.


6. Format the letter so that it is nice and big. Here are some hints…. 

label.setFont(label.getFont().deriveFont(28.0f));
label.setHorizontalAlignment(JLabel.CENTER);

7. Add a key listener to the frame in setup method. Add label to the panel and panel to the frame.


8. Create object/instance of typingTutor class in main method. Using the object/instance call the setup method as shown below. 

  main method(){
typingTutor tutor = new typingTutor();
tutor.setup(); 
 }


9. Make the letter change every time a key is pressed. To do that, in the keyReleased method:
      i) Reset currentLetter.
      ii) Update your JLabel.


10. In the keyPressed method, print out the character that the user typed.


11. If they typed the currentLetter, print “correct”.

12. If they typed the correct letter, set the panel background to green. Otherwise set it to red.

13. [optional] After a certain number of characters, show the users’ typing speed.


Date timeAtStart = new Date();
     
private void showTypingSpeed(int numberOfCorrectCharactersTyped) {
    Date timeAtEnd = new Date();
    long gameDuration = timeAtEnd.getTime() - timeAtStart.getTime();
    long gameInSeconds = (gameDuration / 1000) % 60;
    double charactersPerSecond = ((double) numberOfCorrectCharactersTyped / (double) gameInSeconds);
    int charactersPerMinute = (int) (charactersPerSecond * 60);
    JOptionPane.showMessageDialog(null, "Your typing speed is " + charactersPerMinute + " characters per minute.");
}
