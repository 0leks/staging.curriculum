Typing Tutor


We are going to make a Typing Tutor .


1. Create a JFrame and a JPanel for your Typing Tutor.


2. Create a member variable in class and initialize it in the main method as shown below:


char currentLetter;
currentLetter = generateRandomLetter();

3. Paste the following method after the main method. 

char generateRandomLetter() {
    Random r = new Random();
    return (char) (r.nextInt(26) + 'a');
}

4.Create a JLabel and set the text of the label to the currentLetter. Make sure it changes every time you start the game.


5. Format the letter so that it is nice and big. Here are some hints…. 

label.setFont(label.getFont().deriveFont(28.0f));
label.setHorizontalAlignment(JLabel.CENTER);

6. Add a key listener to the frame.


7. Make the letter change every time a key is pressed.

To do that, in the keyReleased method:
      i) Reset currentLetter.
      ii) Update your JLabel.


8. In the keyPressed method, print out the character that the user typed.


9. If they typed the currentLetter, print “correct”.

10. If they typed the correct letter, set the panel background to green. Otherwise set it to red.

11. [optional] After a certain number of characters, show the users’ typing speed.


Date timeAtStart = new Date();
     
private void showTypingSpeed(int numberOfCorrectCharactersTyped) {
    Date timeAtEnd = new Date();
    long gameDuration = timeAtEnd.getTime() - timeAtStart.getTime();
    long gameInSeconds = (gameDuration / 1000) % 60;
    double charactersPerSecond = ((double) numberOfCorrectCharactersTyped / (double) gameInSeconds);
    int charactersPerMinute = (int) (charactersPerSecond * 60);
    JOptionPane.showMessageDialog(null, "Your typing speed is " + charactersPerMinute + " characters per minute.");
}
