Cuteness TV


Goal: Make a program using Swing and Model-View-Controller design architecture.

The program will have a window (frame) with a number of buttons that will show different videos when clicked. There will be FOUR Java classes to complete this program.


    Make the View


  1. This class will have a frame, a panel, and some buttons. NOTE: because this GUI has more than one button, you must add a JPanel to your JFrame and put the buttons on the panel! The View will need to have a variable for the controller (because it will need a mouse listener).
  2. The runner class needs a main method and will create the view object when the program is run.

    3. Connect a Controller


  3. Make the controller a mouse listener so it can process mouse clicks.
  4. Connect the controller to the model.
  5. Add code to the controller for each of the buttons on the view.

    6. Complete the Model
