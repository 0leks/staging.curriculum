Sound Effects Machine


Goal: Make a program using Swing and Model-View-Controller design architecture.

The program will have a window (frame) with a number of buttons that will play different sound effects when clicked. There will be FOUR Java classes to complete this program.


    Make the View


  1. This class will have a frame, a panel, and some buttons. NOTE: because this GUI has more than one button, you must add a JPanel to your JFrame and put the buttons on the panel! The View will need to have a variable for the controller (because it will need a mouse listener).
  2. The runner class creates the view object when the program is run

    3. Connect a Controller


  3. Make the controller a mouse listener so it can process mouse clicks. The controller acts as the go-between so has variables for the view and model
  4. Add code to the controller for each of the buttons on the view.

    5. Complete the Model


  5. A sample sound effect has been provided. Add some more of your own here.

    They must be .wav files. Free sound effects can be found here. The League also has an account at freesound. Ask your teacher for our login.

    The .wav files must be placed into your sound effects recipe package (drag and drop).
    6. If you need help, take a look at the previous recipes in this module. Drum Kit is a good one to look at since it also plays sound files and will have a lot of methods you can use.
