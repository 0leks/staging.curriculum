Bieber Fever

Goal:

Print the lyrics of this song using a for loop and if statements!

Steps:

  1. Write a program that prints these lyrics from Justin Bieber's song, "Baby".

    HOWEVER: Your code can only contain one "like", one “baby”, and one “oh”, so you will need a for loop and some if statements. 
    And I was like baby, baby, baby oh
Like baby, baby, baby no
Like baby, baby, baby oh
I thought you'd always be mine

Baby, baby, baby oh
Like baby, baby, baby no
Like baby, baby, baby ooh
I thought you'd always be mine
  2. Notice, every 2nd line the “oh” should be “no”
  3. Each chorus ends with “I thought you'd always be mine”.
