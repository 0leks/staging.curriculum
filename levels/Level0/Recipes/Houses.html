Houses

Goal:

Use the Robot to draw a skyline.

Steps:

  1. This recipe practices writing and calling void methods that take parameters.
  2. Create a robot.
  3. Have a robot start on the left side of the window near the bottom.
  4. Draw a flat-topped house of height 100 with grass after it (see picture).
  5. Use a for loop to draw 10 flat-topped houses.
  6. Change the code to make the height of each house random (100-500) so the loop will now draw 10 houses of different heights.
  7. Make each house draw in a random color.
  8. Set the scene to night. (Hint: use Robot's setWindowColor() method).
  9. Give houses peaked roofs if their height is less than 30.Taller houses keep flat roofs.
