increment

Know how to increase or decrease number variables

int count = 5;     // declare and initialize a variable
Java has a shorthand for adding 1 or subtracting 1 from an int as follows: 
count++;   //increase count by one
count--;   //decrease count by one
If you want to increase or decrease by more than one, you have a choice of formats: 
count = count + 5;   //increase count by 5;
count += 4;          //increase count by 4;

count = count - 3;   //decrease count by 3;
count -= 2;          //decrease count by 2;
