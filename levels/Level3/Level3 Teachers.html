MODULE 0

Creating, using, and iterating through arrays of all types

Recipes
Code     bit.ly/l3_mod0
Solutions

At the end of this module, students will be able to:

       • Create an array of any data type or object
       • define the size of an array with new[]
       • determine the length of a pre-defined array
       • access the first and last element in an array
       • iterate through the elements of an array using a for loop
       • initialize an array with constant values

MODULE 1

Writing and using common data structures

Recipes
Code     bit.ly/l3mod1
Solutions

At the end of this module, students will be able to:

       • Identify the member variables and methods that make up a dynamic array data structure
       • Use the java.util.ArrayList class
       • Identify the member variables and methods that make up a stack data structure
       • Use the java.util.Stack class
       • Identify the member variables and methods that make up a map data structure
       • Use the java.util.HashMap class
       • Know that Java's collections can only collect Objects and not primitive data types
       • Give the definition of a wrapper class

MODULE 2

Identifying and writing common sorting algorithms

Recipes
Code     bit.ly/l3mod2
Solutions

At the end of this module, students will be able to:

       • sort an array of numbers using a bubble sort
       • give the definition of recursion in a programming sense
       • identify which sorting algorithms use recursion
       • find the location of an element in an array using a linear search
       • identify the data pre-requesites for a binary search, interpolation search, and exponential search
       • compute the total of an array of numbers
       • find the smallest and largest element in an array of numbers

MODULE 3

Using Interfaces and Abstract classes

Recipes
Code     bit.ly/l3mod3
Solutions

At the end of this module, students will be able to:

       • create abstract classes and interfaces
       • make a java class extend an abstract class
       • make a java class implement an interface
       • know the definition of subclass and superclass
       • identify absract methods
       • override absract methods
       • know that abstract methods must be overridden by an inheriting class
       • know that abstract classes and interfaces cannot be directly instantiated
       • know that only one class can be extended
       • know that multiple interfaces can be implemented
