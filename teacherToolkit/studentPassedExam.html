Student Passed an End-of-Level Exam

Once a student has taken (and passed) an end-of-level exam, there are a few administrative steps to take:




Presentation to Student


  • Present the student with the lanyard pin for the level they passed.
    (Please ask a member of staff if you do not know where these are to be found)
  • Take a photo of the student and email it to The League admin
  • Ask the student if they would like to receive a printed certificate.


Register the Event with The League


  • Enter the student details, level passed, grade, certificate wanted, etc. via the Teacher Quick links page

  • The above process should trigger moving the student into a higher level class. If by the following week this has not happened, please email The League admin


Move the Student Code to their GitHub


While the student is working in a particular League level, their code is managed by The League's GitHub organization.

For example: in Level 0, the code is managed by League-Level0-Student

Before the student starts working on the next level, they should save the code they have written to their own GitHub as described in the following Tutorial
